Sample: Quick NPV in Python


import numpy as np
cash_flows = np.array([150, 170, 190, 210, 230])  # in millions
discount = 0.09
npv = np.sum(cash_flows / (1 + discount) ** np.arange(1, len(cash_flows) + 1))
print(round(npv, 2))
Explain discounting intuition: future cash flows worth less today; relate to hurdle rate.

Key Financial Formulae

NPV: $$\mathrm{NPV}=\sum_{t=1}^{T}\frac{CF_t}{(1+r)^t}-I_0$$

CAPM: $$\mathbb{E}[R_i]=R_f+\beta_i\big(\mathbb{E}[R_m]-R_f\big)$$

NPV guides project selection; CAPM links expected return to systematic risk (beta).
